# ZenBot AI - Intelligent Chatbot
## CodeAlpha Java Programming Internship - Task 3

### Description
ZenBot is an AI-powered chatbot built with Java Swing, 
connected to the Groq API using Llama 3 model for 
real intelligent responses.

### Features
- Real AI responses powered by Groq + Llama 3
- NLP: Tokenization, Stop Word Removal, Stemming
- Confidence Scoring for intent detection
- Beautiful dark theme chat UI
- Conversation memory (remembers context)
- Typing indicator while AI is thinking
- Quick reply chips for common questions
- Send button hover effects

### Technologies Used
- Java 11+
- Java Swing (GUI)
- Java HttpClient (API calls)
- Groq API (Llama 3.3 70B model)
- NetBeans IDE

### How to Run
1. Clone this repository
2. Open in NetBeans IDE
3. Get a free API key from console.groq.com
4. Add your key in chatbotApp.java
5. Run the project (F6)
